# Harvest Printing Centre Website

A professional business website created for **Harvest Printing Centre** to showcase printing services, provide customer information, and allow clients to make bookings online.

## 🌐 Live Website
https://harvest-printing-centre.vercel.app

---

## 📌 Project Overview

This website was developed to give Harvest Printing Centre an online presence where customers can:

- View available services
- Learn more about the business
- Make bookings online
- Contact the business easily
- Explore the company portfolio

The website is fully responsive and designed with a clean, modern layout.

---

## ✨ Features

- Responsive Design
- Home Page
- Services Page
- Portfolio Section
- Booking System
- Contact Page
- Smooth Navigation
- Custom Styling with CSS
- Interactive JavaScript Features
